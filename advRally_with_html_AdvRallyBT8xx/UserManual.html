AdvRallyBT — Adventure Rally & Day Use Bluetooth Controller

DIY Bluetooth handlebar controller for rally roadbook, navigation, and everyday media control.

BLE Keyboard (HID) ESP32 Stamp-Pico (M5) On-device Wi-Fi setup

🔗 Project site: hagayb.wixsite.com/advrallybt

Purpose

The AdvRallyBT controller is built for motorcycle riders seeking simple, reliable handlebar control.

✅ Primary: Rally roadbook apps (e.g., Piste Roadbook on Android, Rally Roadbook on Apple tablet)
🎯 Secondary: Navigation apps (e.g., OsmAnd) · Everyday: Media & Zoom

Versions

  • Enduro: Left-handlebar mount, 3 buttons + 1 toggle (5 functions)
  • Adventure: Under left-handlebar, 1 button + 2 toggles (5 functions)
  • Adventure Pro: Under left-handlebar, 1 joystick + 1 toggle + 1 button (8 functions)

Technology

  • Controller: Arduino ESP32 Stamp-Pico (M5)
  • Bluetooth Profile: BLE Keyboard (HID)
  • Configuration: Built-in Wi-Fi Access Point with setup page
  • Quick changes: Key combinations on the unit

Quick Start

  1. Power On: Connect USB power. Status LED slow-blinks when Bluetooth is not connected. A temporary Wi-Fi AP starts for 2 minutes.
  2. Pair over Bluetooth: On the phone/tablet, scan and select AdvRallyBT8xx (xx = unit number). Accept prompts (iOS may require confirmation). RGB LED shows BLUE when connected.
  3. Wi-Fi setup page (optional): Connect to AP while active:
    • SSID: AdvRallyBT8xx   Password: 12345678
    • Open http://192.168.4.1/
    • Set Active Mode (1–3), update key mappings (incl. second push), reset/restore defaults, and adjust LED brightness.

⚠️ The AP auto-turns off after 2 minutes of inactivity. You can bring it back using a key combination (see On-the-Road Configuration).

Default Button Mapping

Enduro (5 functions)

  • Top Button Center Button Bottom Button
  • Toggle Up Toggle Down

Adventure (5 functions)

  • Main Button
  • Toggle 1 Left/Right
  • Toggle 2 Up/Down

Adventure Pro (8 functions)

  • Joystick: Up, Down, Left, Right, Push
  • Toggle Up / Toggle Down, Button

Three Operating Modes & Default Actions

Mode 1 — Media / Zoom

  • Up/Down = Volume · Left/Right = Previous/Next track
  • Push = Play/Pause
  • Toggle Up/Down = Volume Up/Down
  • (Pro) Extra Button = Mute

Mode 2 — Navigation / OsmAnd

  • Up/Down/Left/Right = Arrow keys
  • Push = Enter (second action = c)
  • Toggle Up/Down = + / -
  • (Pro) Extra Button = d

Mode 3 — Customizable

Pre-mapped to demo/test keys (e.g., B, T, &, etc.). Easily remap in the Wi-Fi UI.

On-the-Road Configuration

  • Enter config mode (hold ≥ 2s):
    • Enduro: Toggle Up + Top Button
    • Adventure: Toggle 1 Left + Main Button
    • Adventure Pro: Joystick Push + Button
  • While LED blinks rapidly:
    • Joystick Right → Next Mode (1→2→3)
    • Joystick Left → Previous Mode (3→2→1)
    • Joystick Push → Restart Wi-Fi AP
    • Joystick Up/Down → LED brightness

Factory Reset

Unpair button (GPIO39) — Hold > 3 s → rapid blink ×10. Clears all Bluetooth bonds and device settings, then restarts.

LED Indicators

  • Main LED: short flash on key release, slow blink when Bluetooth is not connected, fast blink in config mode, 1/2/3 blinks confirm Mode 1/2/3.
  • RGB SK6812: Blue = BT connected Green = Wi-Fi AP active Red = recent web activity

Pinout / Wiring Reference (ESP32)

GPIOFunction
GPIO26Up / Top Button
GPIO18Down / Bottom Button
GPIO21Left / Center Button
GPIO19Right
GPIO22Push
GPIO25Main LED (PWM)
GPIO27RGB LED (SK6812 DIN)
GPIO39Unpair (input-only)
GPIO01Toggle Up (Pro 8 functions only)
GPIO03Toggle Down (Pro 8 functions only)
GPIO00Extra Button (Pro 8 functions only)

Notes for Apps

  • Roadbook apps: Customize Mode 1 mappings to match Piste/Rally Roadbook key bindings.
  • Navigation (OsmAnd): Mode 2 provides arrows and Enter; adjust to taste in the web UI.

Troubleshooting

Can’t see the setup page?
  • Connect to AP AdvRallyBT8xx and open http://192.168.4.1/.
  • If the AP timed out, power-cycle or use the key combo to restart Wi-Fi AP.
No Bluetooth actions?
  • Verify the device is paired and connected (RGB shows Blue).
  • Some apps only receive keys when they’re in focus.
  • Requires Apple OS / Android 13 or higher.
Factory reset

Hold GPIO39 > 3 s → clears bonds and settings, then reboots.
